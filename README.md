# CodeAlpha_Iris-Flower-Classification.
Machine learning project to classify Iris flower species using Scikit-learn (CodeAlpha Internship)
# Iris Flower Classification

## Overview
This project uses a Decision Tree Classifier to classify Iris flowers into three species:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

## Dataset
The dataset contains 150 Iris flower samples with four features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Libraries Used
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Results
The model was evaluated using:
- Accuracy
- Classification Report
- Confusion Matrix
